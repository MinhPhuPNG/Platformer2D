# 2D Flower Pot Platformer

2D platformer with movement, jumping, double jump, enemies, water, collectible sunflower, and a finish goal.

## Project Setup

1. Clone or download this repository.
2. Open the project using Unity Hub.
3. Open the `Scenes` folder.
4. Run `Level 1` to start the level.

---

## Controls

| Action | Key |
|---|---|
| Move Left | A / Left Arrow |
| Move Right | D / Right Arrow |
| Jump | Space |
| Reset Game | Space |

---

## Folder Structure

| Folder | Purpose |
|---|---|
| Scenes | Unity scenes |
| Scripts | Gameplay scripts |
| Sprites | Player, enemy, background, and object sprites |
| Tiles | Tilemap assets and tile palettes |
